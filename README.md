# sudoku-frontend

Multiplayer sudoku built with ReactJs and Socket.io.

[Backend](https://github.com/mDeram/sudoku-backend)

## Running the project in development
```sh
npm run dev
```

## Running the project in production
```sh
npm run build
# serve the dist/ folder
```
